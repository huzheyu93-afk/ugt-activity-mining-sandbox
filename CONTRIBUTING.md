# Contributing

Thanks for improving UGT Activity Mining Sandbox.

## Good Contributions

- New mocked event types for games, marketplaces, quests, NFT checks, and developer integrations.
- Clearer validation rules for duplicate, low-confidence, or capped actions.
- Safer examples for public campaign rules.
- Tests or fixtures that keep the demo deterministic.

## Avoid

- Adding production wallet keys, private API credentials, or custody logic.
- Adding token reward distribution.
- Adding ROI, yield, price, or guaranteed reward calculations.
- Adding spam automation, mass messaging, or fake engagement tooling.

## Local Check

```bash
npm run check
```
