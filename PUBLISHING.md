## Launch Post

```text
I made a small open-source UGT Activity Mining Sandbox.

It takes mocked wallet, game, marketplace, NFT, developer, and community events and turns them into a contribution score.

The goal is to move "mining" copy away from spam-to-earn and toward useful activity:

- wallet login
- game completion
- marketplace order
- NFT check
- developer API test
- tutorial contribution

No production API, no wallet keys, no token rewards, no ROI calculator.
```

## Community Reply

```text
This is intentionally a sandbox, not a reward contract.

The useful point is the scoring shape: event validation, duplicate checks, confidence, action caps, and transparent risk language. It is meant to help teams publish safer UGT activity mining mechanics before they run any real campaign.
```
